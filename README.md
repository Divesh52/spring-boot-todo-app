**Spring Boot To-Do App**
A simple, yet powerful To-Do application built with Spring Boot, Java, and Maven.
This project is a hands-on exploration of building a backend-driven web application, managing
tasks efficiently, and practicing Spring Boot fundamentals.

**Features**
1. Add, view, and manage tasks
2. CRUD operations on tasks via a clean, minimal UI
3. Organized service, controller, and repository structure
4. Maven-based project for easy setup and dependency management
